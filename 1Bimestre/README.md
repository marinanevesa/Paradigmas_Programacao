# Paradigmas de Programação 1 Bimestre: POO com Java

Repositório dedicado ao estudo de **Paradigmas de Programação Orientada a Objetos (POO)**, desenvolvido durante o 3º semestre de Sistemas de Informação na **Uni-FACEF**.

## 📂 Estrutura do Projeto

O projeto está dividido entre atividades práticas realizadas em aula e listas de exercícios complementares desenvolvidas em casa:

### 🏫 Atividades em Aula
* [**ClassesObjetos**](./ClassesObjetos): Conceitos fundamentais de classes, atributos e instanciamento.
* [**Encapsulamento**](./Encapsulamento): Práticas sobre visibilidade (getters/setters) e proteção de dados.
* [**TodoParte**](./TodoParte): Primeiros passos na lógica de associação entre objetos.
* [**Heranca**](./Heranca): Implementação de reaproveitamento de código e especialização.

### 🏠 Listas de Exercícios 
Pasta centralizada em [**ListasExercicios**](./ListasExercicios), contendo os desafios propostos pelo professor:

* [**Lista 1**](./ListasExercicios/Lista1): Foco em Classes e Construtores.
* [**Lista 2**](./ListasExercicios/Lista2): Práticas focadas em Encapsulamento.
* [**Lista 3**](./ListasExercicios/Lista3): Estudo detalhado sobre tipos de Relacionamentos:
    * Agregação, Composição e Associação.
* [**Lista 4**](./ListasExercicios/Lista4): Tópicos avançados de Polimorfismo e Especialização de Comportamento.
* [**Exercicio_TodoParte**](./ListasExercicios/Exercicio_TodoParte/src): Exercícios de fixação sobre o conceito de Todo/Parte.
