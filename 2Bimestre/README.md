# Paradigmas de Programação: 2º Bimestre - Spring Boot

Este módulo é focado na transição do Java Desktop/Console para o desenvolvimento **Backend Web**, explorando a criação de APIs (Application Programming Interfaces).

## 🎓 Conteúdo Estudado

### Fundamentos de API REST
* **Conceito de API:** Atuando como um "garçom" entre sistemas.
* **Protocolo HTTP:** Uso de verbos para operações de dados:
    * `GET`: Recuperar informações.
    * `POST`: Criar novos recursos.
    * `PUT`: Atualizar recursos existentes.
    * `DELETE`: Remover dados.

## 🛠️ Como rodar o projeto
1. Certifique-se de ter o Maven instalado.
2. Navegue até a pasta: `cd 2Bimestre/projeto-memoria` (ou o nome da sua pasta Maven).
3. Execute: `mvn spring-boot:run`
