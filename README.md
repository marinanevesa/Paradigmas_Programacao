# Paradigmas de Programação: POO com Java 

Repositório dedicado ao estudo de **Paradigmas de Programação (POO)**, desenvolvido durante o 3º semestre de Sistemas de Informação na **Uni-FACEF**.

## 📂 Estrutura do Projeto

O projeto está organizado por tópicos e listas de exercícios para facilitar a navegação:

* **[ClasseObjeto](./ClasseObjeto):** Introdução aos conceitos fundamentais de classes e instanciamento de objetos.
* **[Encapsulamento](./Encapsulamento):** Práticas sobre visibilidade de atributos e métodos (getters/setters).
* **[Lista1](./Lista1):** Primeira bateria de exercícios de fixação.
* **[Lista2](./Lista2):** Segunda bateria de exercícios focada em lógica e estrutura de classes.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Java
* **IDE:** IntelliJ IDEA
* **Controle de Versão:** Git & GitHub

## 🚀 Como executar
Para rodar qualquer um dos exercícios:
1. Clone este repositório.
2. Abra o projeto no **IntelliJ IDEA**.
3. Localize o arquivo que contém o método `main` (ex: `TestaDrone.java`).
4. Clique com o botão direito e selecione **Run**.
