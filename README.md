# Paradigmas de Programação: Java e Spring Boot

Este repositório contém todos os projetos e estudos desenvolvidos durante o 3º semestre do curso de Sistemas de Informação na **Uni-FACEF**. O foco é o aprendizado de arquiteturas modernas e paradigmas de desenvolvimento.

## 📂 Estrutura do Repositório

O repositório está organizado por etapas letivas:

* **[1º Bimestre](./1Bimestre):** Fundamentos de Programação Orientada a Objetos (POO) com Java Puro.
* **[2º Bimestre](./2Bimestre):** Desenvolvimento de APIs RESTful utilizando o framework Spring Boot.
